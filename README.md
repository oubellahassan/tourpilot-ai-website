# TourPilot AI — Promotional Website

> AI-powered tour operator & DMC software. Build itineraries, manage suppliers, automate invoicing, and close more bookings from one intelligent platform.

## 🌐 Live Site

**Domain:** [tourpilotai.com](https://tourpilotai.com) *(pending setup)*

## 🚀 Deployment

This is a static HTML/CSS/JS site — no build step required.

### Option 1: Netlify (Recommended)
1. Push this repo to GitHub
2. Go to [netlify.com](https://netlify.com) → New site from Git
3. Select this repo → Deploy
4. Add custom domain: `tourpilotai.com`

### Option 2: Vercel
1. Push to GitHub
2. Import on [vercel.com](https://vercel.com)
3. Framework preset: `Other`
4. Output directory: `.`
